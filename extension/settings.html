How to Get Your Canvas API Key

  1. Go to your Canvas Student Account
  2. Select Settings
  3. Under the "Approved Integration" Section, Click: New Access Token
  4. For reasoning put: "Focus Extension"
  5. Click Generate Token
  6. Copy the entire token and paste it below
Your institution's Canvas URL (default is correct for most users)
Your API key will be stored locally and used only to access your Canvas courses
